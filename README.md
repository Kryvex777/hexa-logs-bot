# 🤖 Hexa Studios - Discord Log Bot

A powerful Discord bot that logs server activities.

## ✨ Features

| Log Type | Description |
|----------|-------------|
| 📝 Message Edits | Track message edits |
| 🗑️ Message Deletes | Track message deletions |
| 📁 Channel Create | Track new channels |
| 🎙️ Voice Join | Track voice joins |
| 🔇 Voice Leave | Track voice leaves |
| 🏷️ Role Create | Track new roles |
| 🎭 Role Assign | Track role assignments |

 ## 📝 Commands

| Command | Description |
|---------|-------------|
| `/set_logs channel:#channel log1:type` | Configure logs |
| `/hexastudios` | Bot information |
